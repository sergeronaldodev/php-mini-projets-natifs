# generateur-mot-de-passe-php-natif
Générateur de mots de passe robuste construit avec PHP natif, HTML et CSS. Utilise random_int() et une logique d'inclusion forcée pour garantir des mots de passe sécurisés et diversifiés (minuscules, majuscules, chiffres, symboles) basés sur les options utilisateur. Démontre la maîtrise des fondamentaux PHP côté serveur.
